## AI Usage v0.1.0

Initial Windows release.

### Highlights

- Windows tray-only app with taskbar hidden
- Claude, Codex, Gemini, Antigravity, and Cursor providers bundled
- Claude, Codex, Gemini, and Antigravity enabled by default
- Cursor bundled but disabled by default
- Global shortcut, 5-minute auto refresh default, start on login default
- Local HTTP API on `127.0.0.1:6736`
- NSIS and MSI installers

### Installers

- `AI Usage_0.1.0_x64-setup.exe`
- `AI Usage_0.1.0_x64_en-US.msi`
